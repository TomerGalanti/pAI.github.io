Mark a piece of code. pAI learns from traces and makes it faster over time.

pAI lets developers mark an expensive function with a decorator. As repeated calls come in, pAI learns the workload, synthesizes specialized replacements, verifies them, and patches only that function. This replay uses PACE 2025 Dominating Set: on 100 private graphs, the selected solver returns valid outputs with average size 231,595 in 2.89s. Released PACE solvers return slightly smaller sets, but take hundreds of seconds. Your application still calls solve()..
